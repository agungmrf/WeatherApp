# WeatherApp-Kotlin

An Android App where you can check weather from city you want and see forecast for next three days, it was built with mvvm, dagger-hilt, livedata,coroutines.

# Features
- Kotlin
- MVVM design pattern
- Retrofit2
- ViewBinding
- Coroutine
- Weather Api
- LiveData
- Retrofit 2
- Dagger-Hilt
